# Install Instructions
Install tensorflow and CUDA Drivers
https://medium.com/@karol_majek/10-simple-steps-to-tensorflow-object-detection-api-aa2e9b96dc94
 
Install Pygame
```py
pip install pygame
```
Install Numpy
```py
pip install numpy
```
Install TelloPy - https://github.com/hanyazou/TelloPy
```py
pip install tellopy
```
Isntall PyAV (FFmpeg)
```py
pip install av
```
